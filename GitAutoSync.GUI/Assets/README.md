# Git Auto Sync GUI

This is the GUI variant of Git Auto Sync.

## Usage

To run the GUI version:

```bash
dotnet run -- --gui
```

Or:

```bash
dotnet run -- -g
```

## Features

- Add and manage repositories
- Start/stop individual repositories or all at once
- View real-time activity logs
- Load configuration from TOML files
- Modern, responsive UI built with Avalonia
